<CardGrid stagger>
	<Card title="Async Support" icon="seti:c">
		Ecewo is built on [libuv](https://github.com/libuv/libuv), a native C library designed for non-blocking asynchronous I/O operations and originally developed for Node.js.
	</Card>

  <Card title="Simple As Express.js" icon="node">
    Inspired by the simplicity of Express.js, it remains just as simple — but offers much more power.
	</Card>

	<Card title="Built-in JSON Support" icon="seti:json">
    Ecewo optionally includes [cJSON](https://github.com/DaveGamble/cJSON), a built-in C library for working with JSON.
	</Card>

  <Card title="Middleware Structure" icon="puzzle">
		Express.js-style route-specific and global middlewares are supported.
	</Card>

  <Card title="Env Configuration" icon="setting">
		Ecewo optionally comes with [dotenv-c](https://github.com/Isty001/dotenv-c), a built-in `.env` file parser for C.
	</Card>

  <Card title="Authentication Mechanism" icon="seti:lock">
		You can easily do authentication and authorization using the build-in session management mechanism.
	</Card>

	<Card title="Read The Docs" icon="open-book">
		Start from [Introduction](/docs/introduction) page to read the documentation.
	</Card>

	<Card title="Star on Github" icon="star">
		Star the [Github repo](https://github.com/savashn/ecewo) if you find it useful.
	</Card>

</CardGrid>
